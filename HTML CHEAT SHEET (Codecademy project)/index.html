HTML basic elements and structures

What is HTML?

HTML stands for HyperTextMarkupLanguage and it is used to create the structure and content of a webpage. Most HTML elements contain opening and closing tags with raw text or other HTML tags between them. Html elements can also be nested inside other elements. The enclosed element is the child of the enclosing parent element.

Basic Structures

Any HTML code must be written inside the html tag, without it, the code wont work.

All metadata must be included in the head tag, which may contain for example the title tag (that shows thet title of the page on the browser's tab) or the link tag (which can connect different files to the original file, such as connecting a css file to a html file.)

Now, for visible content there is a tag called body in which all visible conten must be encapsulated. For headings and sub headings we can use the h1 though h6 elements which are used to enlarge text. The p , span , and div tags are used to specify text or blocks of text.

In HTML a space on the code editor is not recognized as a space for the webpage itself, since they are used to organize the code and make it more readable. So, to create line breaks on our webpage we use the br tag.

In HTML we also have two different types of list. Ordered list are numbers and are created with the ol tag and Unordered list are bulleted and created with the ul tag. It is important to remember that inside of both of these tags we must nest a li tag that will represent the item or the list and allow it to be displayed.

Images and videos can also be added by linking to an existing source using the self-closing img tag for images and the video tag.

Another useful tag in HTML is the a tag. You can create sections on a webpage and jump to them by adding ids to the elements you wish to jump to.

Finally, comments are written in HTML using the following syntax (real comments use less than signs at the beggingn and greater than sign at the end):
!-- I would be a comment if I had my initial tags--

Important note*: All tags must start with less than signs at the beggingn and greater than sign at the end.

HTML Tables

Html tables can be created with the table tag. To add a row to a table the tr tag can be used. To ad data to a row, the td tag can be used. The headings can be added with the th element. The data in the tables can span columns using the colspan attribute and it can span rows using the rowspan attribute.

To mantain order in your tables you might want to split your table in three main sections: the head, a body, and a footer. A tables head is created with the thead element. A tables body is created with the tbody element. Finally, a tables footer is created with the tfoot element.

CSS Setup and Selectors

What is CSS?

CSS is one of the programming language that can change the look of HTML elements. In order to do this, CSS must select HTML elements, then apply the different styles to them.

CSS can select HTML elements by their tag, class, or ID. It is possible to apply multiple CSS classes can be applied to one HTML element. Styling classes can be reusable, while IDs can only be used once. IDs are more specific than classes, and classes are more specific than tags. That means IDs will override any styles from a class, and classes will override any styles from a tag selector.

Multiple selectors can be chained together to select an element. This raises the specificity, but can be necessary. Nested elements can be selected by separating selectors with a space. Multiple unrelated selectors can receive the same styles by separating the selector names with commas. There is a flag called !important that will override any style, however it should almost never be used, as it is extremely difficult to override.

CSS Visual Rules

CSS declarations are structured into property and value pairs. There are various basic properties to style the different HTML elements. This are some examples:

  • The font-family property defines the typeface of an element.
  • The font-size property controls the size of text displayed.
  • The font-weight property defines how thin or thick text is displayed.
  • The text-align property places text in the left, right, or center of its parent container.
  • Text can have two different color attributes: color and background-color. color defines the color of the text, while background-color defines the color behind the text.
  • The opacity property can make an element more or less transparent.
  • The backgroung-image property can set the background of an element to an image.
